| 身体动觉

 | 22 | 身体动觉：用身体与他人进行互动，使用或调动人的某些身体功能。主要表现在健身运动、形象装扮、戏剧表演或者中医诊疗等领域。身体动觉天赋的代表人物是擅长“望闻问切”的神医扁鹊，善歌舞的杨玉环，京剧大师梅兰芳。典型行业有体育、时尚、表演、美容和中医。 |

| 操控实践 | 31 | 操控实践：用身体与外在事物之间的互动，表现在手工制作、乐器演奏或者劳动生产等方面。操控实践天赋的代表人物有：木匠鼻祖鲁班、弹琴技艺高超的俞伯牙。典型行业有手工艺、乐器演奏、服装设计、劳动生产和体育运动等。 |
| 本能体验 | 61 | 你的本能体验这项能力比较强，对于信息的敞开和接受度较高，喜欢通过真实的体验来认识世界，擅长接收和表达当下的信息，因此在文化旅游、灵性感知或者艺术创造等方面拥有的天赋水平比较高。本能体验天赋代表人物是自在超脱、云游四方的老子，精通诗画的唐伯虎和诗仙李白。典型行业有身心灵、艺术创作、体验式培训和文化旅游等。 |
| 人际关系 | 40 | 人际关系：连接、整合或者协调人与人之间的关系，表现在人际交往、心理共情或者组织协调等方面。人际关系天赋的代表人物：宣扬仁爱思想的孔子，知人善用的刘备，慷慨待人的宋江。典型行业：心理咨询、公共关系、家庭教育、团队管理等。 |
| 事物统筹 | 51 | 你的事物统筹能力比较强，擅长驾驭事物与事物之间的关系，与事物建立情感连接，组织事物，协调事物，或者流通事物等，因此在生活美学、系统组装或者物流采购等方面拥有的天赋水平比较高。事物统筹天赋代表人物：治水的大禹，建筑学家贝律铭，茶圣陆羽。典型行业：茶艺师、插花师、物流采购、室内设计和建筑设计等。 |
| 信息整合 | 22 | 信息整合：驾驭信息与信息之间的关系，用信息传达感情，接收信息中的感性传达，整合信息，或者传递信息等，表现在传媒、文学、通讯或者商业等方面。信息整合天赋代表人物：王阳明、巴菲特、汪涵。典型行业：互联网、金融投资、文化传播、通讯和媒体等。 |
| 人物洞察 | 36 | 人物洞察：对人当下的状态关注度，观察人、研究人，表现在察言观色、识人断人或者人才管理等方面。人物洞察天赋的代表人物：鬼谷子，曾国藩，张爱玲典型行业：人事招聘、管理、商务谈判，小说创作和心理咨询等 |
| 事物研发 | 16 | 事物研发：对事物进行观察和研究分析，表现在自然科学、科技发明或者工艺制造等相关领域。事物研发天赋的代表人物有：物理学家牛顿、发明家爱迪生、中药学家李时珍。相关的行业有科技发明、工艺制造和生物研发等。 |
| 抽象思维 | 50 | 你的抽象思维能力比较强，比较擅长对信息内容进行理解、联想和分析，所以在逻辑推理、哲学思考或者战略规划等相关领域拥有的天赋比较高。抽象思维天赋的代表人物：诸葛亮、蒲松龄、爱因斯坦典型行业有教育培训、理学研究、策划和战略规划 |
